## рџ“Њ About the Project

This project allows users to explore the noble houses from **Game of Thrones** by selecting a region from a dropdown menu.

The application retrieves house data from the **Ice and Fire API** and dynamically displays relevant information about each house.

Displayed information includes:

- House name
- Region
- Coat of arms
- Words (motto)
- Titles
- Seats

The interface is inspired by the visual style of the series and designed to be **responsive and user-friendly**.
